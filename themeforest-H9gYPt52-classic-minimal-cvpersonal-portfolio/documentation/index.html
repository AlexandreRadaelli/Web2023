Section Header

You can change the background image by modifying the style tag (cover.jpg) at assets/images

Change first letter of your name in (.box) class and tagline (.tagline) 

<section id="home" class="header" style="background-image: url('assets/images/cover.jpg')"> <!-- change image here -->
<div class="sector">
<div class="box">
<h1> J </h1> <!-- change name first char -->
</div>
<h1 class="main-title"> John Doe </h1><!-- /. name -->
<h4 class="tagline"> Web | UI designer </h4><!-- /. profession -->
</div>
</section> <!-- /. header/cover -->

Skill and Education Section

Carousel is used to display skills

  • Skill sections are identified (.skill-area)
  • Each skill items are marked in html file, change with yours (maintain the structure)
  • Each group of skills has major category title (.skill-title)
  • Change percentage and skill name

Skill Bar (within each skill-area) and within skill-holder(owl-carousel item)

 </div>
 <div class="skillbar-bg">

</div> <!-- /.skill bar -->
